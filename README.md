## 核心功能拆解

### 多路径同时备份

这是最核心的一个功能。

以前拷卡，要先拷到硬盘 A，再拷到硬盘 B，一来一去浪费双倍时间。KOCARD 支持同时向多个磁盘路径备份，插两块硬盘，勾选两个目的地，点开始——两块盘同时写入。

而且你可以把这些目的地路径保存成预设。下次拍完直接选预设开拷，省去每次重新配置的麻烦。

### 文件哈希校验——拷完就知道有没有坏

这个功能，可能是 KOCARD 里最让专业用户安心的一个。

它会在拷贝前后分别对每个文件做哈希函数校验，说白了就是给文件算一个"指纹"，对比拷前拷后是不是一样。不一样，就说明拷坏了——然后软件会自动重新拷这个文件，直到通过为止。

有多少次你拷完以后格掉了卡，后来发现有文件打不开？有了这个功能，至少拷完那一刻你是确定安全的。

### 自定义转码

KOCARD 集成了转码功能，内置三种场景预设：

- **代理制作**：生成小码率代理文件，方便剪辑
- **存档备份**：转码成稳定格式长期保存
- **分发预览**：输出适合发送、审片的轻量版本

当然也可以自定义转码参数，满足更复杂的工作流。

### 拷卡完成通知推送

素材量大的时候，拷一次可能要几十分钟，没必要一直盯着屏幕。

KOCARD 支持短信和邮件通知，拷完了或者出错了，直接推到手机。你可以去喝杯咖啡、和导演对下次方案，不用守着电脑。

### PDF 报告导出

拷完可以导出详细的 PDF 报告：哪些文件、用了多长时间、校验结果如何，按日期筛选，还能生成周报。

对于需要向客户或制片方汇报素材状态的人，这份报告直接发过去，专业感直接拉满。
